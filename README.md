# Projeto de Estimativa de Tempo de Serviço na Agricultura

Projeto de Ciência de Dados voltado à área agrícola, desenvolvido em Python através do Google Colab.

<br>

**Descrição da Tarefa:**

Dado um dataset contendo diferentes ordens de serviço, o problema consiste em estimar o tempo que um pulverizador finalizará uma nova ordem de serviço. 

<br>

**Resolução:**

O problema foi tratado a partir de análise exploratória de dados e contrução de modelos de regressão para predição do tempo de uma ordem de serviço (variável resposta). Informações mais detalhadas acerca do problema e de sua resolução computacional podem ser obtidas no [*notebook*](https://github.com/llcorrea/agriculture_time_prediction/blob/main/TesteDados.ipynb) presente nesse repositório.
